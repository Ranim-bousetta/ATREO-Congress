# ATREO Congress Website

Responsive React + Vite + Tailwind CSS website inspired by the uploaded presentation.

## Pages included

- Home
- Speakers
- Program
- Sponsors
- Register

## Run locally

```bash
npm install
npm run dev
```

## Build for production

```bash
npm run build
npm run preview
```

## Notes

- The site uses the images extracted from your PowerPoint.
- The registration form is front-end only for now.
- Some speaker names are placeholders because the PPT did not include every individual speaker file.
- Replace any asset inside `src/assets/` and update `src/data/siteData.js` whenever you have the final folders for speakers and sponsors.
